# Logos de marcas

Coloca aqui los logos pequenos que se muestran en el selector de marca.
Ahora hay monogramas provisionales para que el selector ya se vea completo; puedes reemplazarlos por los logos oficiales manteniendo estos mismos nombres.

Especificaciones recomendadas:
- Formato ideal: SVG transparente.
- Alternativa: PNG transparente de 512x512 px.
- Usar version blanca o clara si el logo original no funciona sobre fondo oscuro.
- Mantener el logo centrado y con aire alrededor.
