# EVINKA AI Calculator

Landing page futurista para EVINKA con React, Vite, TypeScript, Tailwind CSS, una escena 3D embebida con Spline y calculadora de ahorro EV.

## Vista publica

Cuando GitHub Pages este activo, el sitio se podra ver en:

https://20223774-hue.github.io/EVinka-Calculator/

## Archivos principales

- `src/components/Navbar.tsx`: logo corporativo EVINKA.
- `src/components/HeroSection.tsx`: hero full-screen con Spline, auto destacado y calculadora.
- `src/components/CalculatorPanel.tsx`: calculadora de kilometros diarios, ahorro anual/mensual y cargador recomendado.
- `src/data/vehicles.ts`: base de marcas y modelos importada del Excel de modelos EV Peru.
- `public/car-images/`: carpeta para imagenes de autos. Ver `README.md` interno para nombres exactos.
- `public/brand-logos/`: carpeta para logos de marca. Ver `README.md` interno para nombres exactos.
- `public/charger-images/`: carpeta para imagenes de cargadores EVINKA.
- `src/styles.css`: variables HSL, Tailwind y estilos globales.
- `tailwind.config.ts`: tema, fuente Sora y animaciones.
