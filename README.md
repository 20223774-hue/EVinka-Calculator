# EVinka Calculator

Calculadora interactiva de ahorro para comparar el costo de uso de vehiculos electricos frente a vehiculos a combustion.

## Vista publica

Cuando GitHub Pages este activo, el sitio se podra ver en:

https://20223774-hue.github.io/EVinka-Calculator/

## Archivos principales

- `index.html`: estructura de la pagina.
- `css/styles.css`: estilos visuales y responsive.
- `js/data.js`: datos de vehiculos y calculos.
- `js/main.js`: interaccion de la calculadora.
- `assets/`: imagenes de vehiculos y cargador EVinka Minibox.
