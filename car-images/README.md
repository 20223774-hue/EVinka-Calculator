# Imagenes de autos

Coloca aqui las imagenes de cada modelo para que aparezcan en la animacion lateral.

Especificaciones recomendadas:
- Formato: PNG transparente o WebP con fondo transparente.
- Tamano recomendado: 1600x900 px o 2000x1100 px.
- Peso recomendado: menos de 700 KB por imagen si es posible.
- Encuadre: auto completo, vista lateral o 3/4, mirando hacia la izquierda o derecha.
- No incluir texto, sombras duras ni fondos blancos.
